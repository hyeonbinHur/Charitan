## Getting Started  

### Client Setup  
1. Navigate to the `client` folder in your terminal:  
```
  npm install
  npm run dev
```
2. Navigate to the `server` folder in your terminal:
```
  npm install
  npm run start
```

3. local redis must be turn on in port 6379
